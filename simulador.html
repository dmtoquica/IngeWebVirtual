1. CONTROLLER (ReservasController)

HTTP HANDLER

Recibe el JSON, extrae los datos y llama al servicio.

createReservation(req) {
  const { roomId } = req.body;
  return service.book(roomId);
}

2. SERVICE (BookingService)

LOGIC

Aplica reglas: ¿La habitación existe? ¿Está libre?

book(id) {
  const room = model.find(id);
  if (room.isOccupied) throw Error(400);
  return model.update(id, true);
}

3. MODEL (RoomModel)

DATA
